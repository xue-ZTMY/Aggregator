
跟踪==使用 git add 添加过==暂存区

git commit

用于将暂存区的更改提交到本地仓库，并生成一个新的提交记录。
-a 
这个选项会自动把所有已经被跟踪（tracked）的文件的修改直接添加到暂存区，无需执行 git add 命令。
如果是新增的文件或删除的文件，仍需手动 git add 或 git rm。
使用 -m 提供提交信息


暂存区：
git status
查看暂存区

git reset HEAD <文件路径>
移除暂存区的文件

本地仓库：
git branch -a
本地和远程分支  

git pull 
远程仓库：
git remote
查看远程仓库
